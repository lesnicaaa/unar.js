> scrapper,这个小机器人是环太平洋2的一个角色，他本身被小女孩寄托着一个拯救社会的梦想。<br>
>就像作者想把unar.js做成前端框架的领跑者一样,为提高javascript开发效率，做出一点点帮助。<br>
> 作者眼中的unar好比环太平洋2中小女孩制造的scrapper
